## Introduce Of Parsers

- `character_parser` : Inven 카드 목록 페이지에서 각 카드 별 캐릭터 정보만 추출하여 `CharacterMainInfo` 테이블에 저장하는 Crawler 입니다.

- `card_parser` : Inven 카드 DB 조회 테이블에서 각 카드에 대한 기본 정보들만 추출하여 `CardInfo` 테이블에 저장하는 Crawler 입니다.
